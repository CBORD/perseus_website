# Onboarding

## First day setup

1. Install Node LTS
2. Run `npm install`
3. Run `npm run dev`
4. Review `docs/architecture.md`

## Branch workflow

- Create feature branch from `main`
- Open PR with clear scope
- Require review before merge

## PR checklist

- [ ] Lint and typecheck pass
- [ ] Page metadata included
- [ ] Structured data added where needed
- [ ] Basic responsive behavior verified

# SEO + GEO Checklist

## Technical

- [ ] Canonical URL for each page
- [ ] Sitemap generated and accessible
- [ ] Robots policy defined
- [ ] Core Web Vitals monitored

## On-page

- [ ] One clear H1 per page
- [ ] Intent-aligned title and meta description
- [ ] Internal links to related pages
- [ ] FAQ blocks where relevant

## Structured Data

- [ ] Organization schema on global layout
- [ ] Website schema on global layout
- [ ] Breadcrumb schema on deeper pages
- [ ] Article schema on blog posts

## GEO

- [ ] Entity-rich factual content
- [ ] Author and source clarity
- [ ] Last updated date displayed
- [ ] Consistent naming across site and social profiles
